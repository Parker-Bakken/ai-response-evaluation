# Quick Checklist (before scoring)

- Did the response answer the question asked (not a different one)?
- Did it respect format/tone/length constraints?
- Did it avoid inventing facts, citations, or data?
- Did it identify missing info and ask for it (if needed)?
- Did it include actionable next steps?
- Did it avoid unsafe medical/legal/financial instructions?
- Is it easy to read and well structured?
